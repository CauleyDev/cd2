My Journey


One fateful day around 4 months ago I decided I wanted to teach myself Python as my first programming language, I followed online material and learned the fundementals, and I realised not only how much fun I was having learning and creating CLI applications, but how easily and quickly I was picking it all up. Knowing I had to make a career move anyway, I made the choice there and then that I would become a developer.

After a lot of research, and recommendations from a UK Developer group I joined on facebook, I purchased Colt Steele's Udemy course "The Web Developer Bootcamp" which teaches you everything you need to know to make a move into a development career. I blazed through the HTML and CSS portions, and I'm currently learning Javascript. Here is my list of things I'm dedicated to learning:



HTML5

Hyper Text Markup Language, the standard language for creating the structure of a web page. Consisting of elements, it tells the browser how to display content.

CSS3
